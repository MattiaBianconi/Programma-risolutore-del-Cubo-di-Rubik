# Programma-risolutore-del-Cubo-di-Rubik
Questo progetto consiste in un software in grado di analizzare la configurazione di un Cubo di Rubik e calcolare la sequenza logica di mosse per risolverlo. Il programma implementa il metodo a strati, introducendo però un'ottimizzazione logica nel primo step risolvendo direttamente la croce bianca saltando il passaggio della margherita.
